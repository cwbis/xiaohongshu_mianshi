## MODIFIED Requirements

### Requirement: 支持追问会话
系统 SHALL 保存 Agent 会话上下文，并在用户追问或进入训练模式时复用原问题、上一轮回答、领域标签和检索来源。

#### Scenario: 训练模式复用已有上下文能力
- **WHEN** 用户在训练模式中完成一轮题目并进入追问或下一题
- **THEN** 系统仍可复用本地检索、历史消息和来源上下文，而不是重新退回无状态问答

### Requirement: 回答来源可追溯
系统 SHALL 在 Agent 回答和训练模式评估中展示用于生成回答或题目的本地来源。

#### Scenario: 训练题与评估都可关联来源
- **WHEN** 系统使用本地帖子、题目或知识点组织训练
- **THEN** 训练题、评估反馈或复盘结果能够关联到对应的来源标题、类型和摘要
