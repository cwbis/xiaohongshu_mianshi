## MODIFIED Requirements

### Requirement: 持久化面试知识 Agent 数据
系统 SHALL 在现有 SQLite 数据库中保存面试题目、知识点、Agent 会话和消息历史，并支持训练模式新增的训练状态与记录。

#### Scenario: 保存训练会话与答题尝试
- **WHEN** 用户进入训练模式并完成出题、作答、评估和追问
- **THEN** 系统持久化训练会话、答题尝试、评估结果和必要的状态字段

#### Scenario: 恢复训练进度
- **WHEN** 用户重新进入已有训练会话
- **THEN** 系统可以恢复当前专题、近期尝试结果和最近一次阶段总结所需的数据
